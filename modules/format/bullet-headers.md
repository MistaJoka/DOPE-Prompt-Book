# Bullet Headers

**Type:** format
**Tags:** #structure #visual #scannable #headers #bullets #organization
**Works With:** all
**Tested On:** claude | gpt-4 | gemini

## Prompt Text

Structure your response using bold headers and bullet points. Each section should have a clear label. Keep bullets concise (1-2 lines max).

## Usage Notes

- **When TO use:** Any response that benefits from scannability — summaries, comparisons, lists of options, multi-part explanations, or reports. Works with virtually any role or task module.
- **When NOT to use:** Conversational or narrative responses where bullets feel choppy. Avoid for short, single-idea answers. If a flowing paragraph is more appropriate, omit this format module or use `format/step-by-step` for ordered content.
